# JavaScript DOM Manipulation

This project focuses on manipulating the DOM using JavaScript. It covers selecting HTML elements, modifying styles and content, handling events, and updating the DOM dynamically without reloading the page.

## Learning Objectives

- Select HTML elements using JavaScript
- Understand the difference between ID, class, and tag selectors
- Modify HTML element styles
- Update HTML content dynamically
- Manipulate the DOM
- Handle DOM and user events
- Make requests using XMLHttpRequest and Fetch API

## Requirements

- All code is written in JavaScript
- Code is executed in a Chrome browser (version 57 or later)
- Code follows semistandard style
- `var` is not used
- DOM manipulation is done without page reload

## Files

- `0-script.js`: Updates the text color of the `<header>` element to red using JavaScript
