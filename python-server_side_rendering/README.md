# Python Server-Side Rendering

## Task 0 - Simple Templating Program

This project demonstrates basic server-side rendering concepts using Python.

### Features:
- Generates personalized invitation files from a template.
- Handles missing data gracefully by replacing it with "N/A".
- Implements input validation and error handling.
- Outputs files sequentially as output_1.txt, output_2.txt, etc.

### Usage:
Provide a template string and a list of attendee dictionaries to the `generate_invitations` function.
