# SQL - More Queries

This project focuses on advanced SQL queries and database management using MySQL 8.0.

## Learning Objectives
- Create and manage MySQL users
- Grant and list user privileges
- Understand PRIMARY KEY and FOREIGN KEY
- Use NOT NULL and UNIQUE constraints
- Retrieve data from multiple tables
- Use JOIN, UNION, and subqueries

## Requirements
- MySQL 8.0
- Ubuntu 20.04 LTS
- All SQL keywords in uppercase
- Each SQL file starts with a comment describing the task

## Files
- `0-privileges.sql`: Lists all privileges for specific MySQL users
