# JavaScript - Warm up

This project contains introductory JavaScript scripts.

## Learning Objectives

At the end of this project, I am able to explain:

- Why JavaScript programming is amazing
- How to run a JavaScript script
- How to create variables and constants
- The differences between var, let and const
- JavaScript data types
- How to use loops and conditional statements
- How to define and use functions

## Requirements

- All files are interpreted on Ubuntu 20.04 LTS using Node.js (14.x)
- The first line of all files is `#!/usr/bin/node`
- All files are executable
- Code follows semistandard style (Standard + semicolons)
